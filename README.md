# Calculadora
## :rocket: Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:
* HTM5
* CSS3  
* JavaScript

## :computer: Projeto
Esse projeto consiste em uma caculadora com as funcionalidades básicas como:
* Somar
* Subtrair
* Dividir
* Multiplicar
* Porcentagem
* Limpar tela ou apenas um número
